# Installation

## Global install

```bash
pnpm add -g docplaybook
```

After installation:

```bash
docplaybook --help
docplaybook init ./my-docs
docplaybook auto ./my-docs
```

## Without global install

```bash
pnpm dlx docplaybook --help
pnpm dlx docplaybook init ./my-docs
pnpm dlx docplaybook auto ./my-docs
```

## Requirements

- Node.js 20+
- A supported model provider
- Markdown docs organized in a local workspace
