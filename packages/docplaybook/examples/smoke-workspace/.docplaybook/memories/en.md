# English Memory

## Terminology
- Translate `知识库` as `Wiki`.
- Translate `租户` as `tenant`.
- Translate `空间` as `space`.

## Style
- Prefer concise technical English.
- Keep UI labels short and direct.

## Protected
- Do not translate CLI commands, file paths, environment variable names, or code identifiers.
- Keep product names in their official English form when known.
