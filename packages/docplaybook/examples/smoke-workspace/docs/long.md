# 长文档批量翻译测试

## 概览

这个页面用于观察 batch translation 的行为。

它应该触发多段落翻译，而不是只有一次单块调用。

## 背景

文档团队希望减少重复翻译，并复用已经确认过的术语。

如果一篇文章很长，系统应该按批次处理，但仍然保持文章级写回。

## 术语

知识库用于沉淀规则。

空间用于组织内容。

租户用于隔离数据。

## 链接

更多信息见 [配置说明](./format-heavy.md)。

## 命令

执行 `docplaybook translate . --force` 可以强制重译。

## 结论

这个页面的重点不是内容本身，而是稳定地产生多个 translatable blocks。
